# test-money

- Приложение работает с TS.
- После работы с приложение желательно очистить localStorage, через консоль 
``` localstorage.clear() ```
Или же через настройки браузера


## Установка проекта

```sh
npm install
```

### Запуск сборки разработки

```sh
npm run dev
```

### Создание сборки для production

```sh
npm run build
```
